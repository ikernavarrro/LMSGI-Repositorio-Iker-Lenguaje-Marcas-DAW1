1 - Página de tributo Una página sencilla dedicada a alguien que admires, ideal para practicar la inserción de imágenes y el formato de texto. 2 - Formulario de encuesta Útil para aprender a dar estilo a los diferentes campos de entrada (input, select, textarea) y botones. 3 - Página de aterrizaje (Landing Page) Una página promocional para un producto o servicio, perfecta para practicar la alineación, los márgenes y el espaciado. 4 - Portafolio personal Crea tu propia página de presentación. Te ayudará a diseñar una cabecera, una sección sobre ti y un pie de página. 5 - Menú de un restaurante Ideal para estructurar listas de elementos con sus descripciones y precios utilizando tipografías atractivas. 6 - Tarjeta de presentación digital Un diseño compacto en el centro de la pantalla que muestre tu foto de perfil, tu nombre y enlaces a tus redes sociales. 7 - Página de inicio de sesión o registro Practica creando contenedores centrados con sombras (box-shadow) y formularios limpios. 8 - Clon de la página principal de Google Un ejercicio clásico para dominar el centrado de elementos y el uso del espacio en blanco. 9 - Diseño de un blog sencillo Estructura una página con un menú de navegación, un área principal para artículos y una barra lateral (sidebar). 10 - Galería de imágenes Una excelente manera de poner en práctica herramientas avanzadas de CSS como Flexbox o CSS Grid. 11 - Tarjetas de precios o suscripción Diseña 3 columnas destacando diferentes planes (básico, pro, premium), ideal para jugar con bordes, fondos y botones llamativos. 12 - Página de error 404 personalizada Deja volar tu creatividad diseñando un mensaje amigable con una ilustración o imagen de fondo cuando una página no se encuentra. 13 - Interfaz de una calculadora básica Dibuja la cuadrícula de botones de una calculadora, un reto estupendo para dominar CSS Grid. 14 - Plantilla de boletín informativo (Newsletter) Diseña un correo electrónico estructurado con un banner superior, un mensaje y un botón de llamada a la acción. 15 - Currículum vitae web (CV) Pasa tu currículum tradicional a código utilizando un diseño limpio de una o dos columnas.
